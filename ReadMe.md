# Introduction to Typescript

## Basics Folder
This has a series of functions that help to explore the type system of Typescript
This folder includes an exploration into the following:

 - Tuples
 - Enums
 - Union
 - Literals
 - Custom Types
 - Return types & "void"
 - Functions
 - "unknown"
 - "never"
